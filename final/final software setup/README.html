Employee Attendence Management System by AccSeedd



Step 1 : Install MYSQL Server 7 or higher


Step 2 : Go to phpmyadmin/sql or your prefered MYSql Interface


Step 3 : Run 'user_creation.sql' (Run directly or copy and paste the code)


Step 4 : Select database 'employee_attendence' -> Go to Import SQL -> Import 'employee_attendence.sql'


Step 5 : Run 'Setup.exe' and Install the System
